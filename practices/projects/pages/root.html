What is Golang?

Golang is a programming language developed by Google.

It was designed to be simple, efficient, and performant, making it especially suitable for systems programming, web development, and cloud-based applications.

Golang Aid commands:

  • go version
  • go help
  • go doc

How to compile Go program?

  • Create a file called main.go
  • touch main.go
  • Build the Go program into an executable file named main
  • go build -o main main.go
  • You should now see a compiled file called 'main'
  • Change the file permissions to make it executable for all users
  • chmod a+x main
  • ./main


  • You should see a great web page accessible in your local browser at http://localhost:10000/
